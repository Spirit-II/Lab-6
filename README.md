## Задание 1. ООП

### Текст задачи 

Разработать класс с тремя целочисленными полями. Создать конструктор копирования.
Разработать метод, вычисляющий максимальное из полей. Перегрузить метод ToString() для
формирования строки из полей класса. Реализовать дочерний класс (его содержание предложить
самостоятельно и описать в решении: какой содержательный смысл имеют поля; реализовать
конструкторы; предложить и реализовать 2-3 метода). Протестировать все конструкторы и
другие методы базового и дочернего классов.

### Алгоритм решения

1) Создать базовый класс с тремя целочисленными полями.
2) Реализовать конструкторы (по умолчанию, с параметрами, копирования).
3) Добавить метод для вычисления максимального из полей.
4) Перегрузить метод ToString() для удобного вывода информации об объекте.
5) Создать дочерний класс с дополнительными полями и методами.
6)  Протестировать все конструкторы и методы базового и дочернего классов.

## Задание 2. Перегрузка операций

### Текст задачи 

Point double x, double y
(координаты точки)
Вычислить расстояние от одной точки до
другой.
Результат должен быть типа double.

### Алгоритм решения

1) Создание класса Piont
2) Создать точку без параметров (Point()) и убедиться, что x = 0, y = 0.
3) Создать точку с параметрами (Point(x, y)) и убедиться, что координаты устанавливаются корректно.
4) перегрузка метода DistanceTo и убедиться, что вывод соответствует формату "Point(x, y)".
5) Создать две точки и вычислить расстояние между ними.
6) Убедиться, что результат совпадает с ожидаемым значением.

## Задание 3. Перегрузка операций

### Текст задачи 

Point 
Унарные операции:
++ увеличить координаты xна 1,
-- уменьшение координаты х на 1.
Операции приведения типа:
int (явная) – результатом является целая часть координаты х;
double (неявная) – результатом является координата y.
Бинарные операции:
+ Point p – вычисляется расстояние до точки p. Результатом должно быть
вещественное число.
+ целое число (лево- и правосторонние операции). Увеличивается
координата х, результатом является объект типа Point.

### Алгоритм решения

1) Создание унарных операций (++, --)
2) Создание приведения типа:
	Явное приведение к int: убедиться, что возвращается целая часть x.
    Неявное приведение к double: убедиться, что возвращается значение y.
3) Создание бинарных операций (+):
    Сложение двух точек: убедиться, что результат — расстояние между ними.
    Сложение точки и числа (лево- и правостороннее): убедиться, что x увеличивается на это число, а объект возвращается корректным.
